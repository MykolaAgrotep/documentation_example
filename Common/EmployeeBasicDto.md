## Purpose  
DTO, содержащий базовую информацию о сотруднике.  
  
## Fields  
  
| Поле       | Тип    | Описание                            |     |
| ---------- | ------ | ----------------------------------- | --- |
| id         | Long   | Уникальный идентификатор сотрудника |     |
| firstname  | String | Имя                                 |     |
| middleName | String | Отчество                            |     |
| lastname   | String | Фамилия                             |     |
