
## Purpose
Хранит **данные сотрудника**, получаемые из API.  
Расширяет [[EmployeeBasicDto]].

---

## Fields

| Name        | Type   | Description                  |
| ----------- | ------ | ---------------------------- |
| id          | Long   | Уникальный идентификатор     |
| firstname   | String | Имя сотрудника               |
| middleName  | String | Отчество (необязательное)   |
| lastname    | String | Фамилия сотрудника           |

---

## Business Meaning

- Используется для **получения данных сотрудника** из API или сервиса.  
- Позволяет **обрабатывать входящую информацию** и отображать её в системе.

---
## Notes

- Расширяет [[EmployeeBasicDto]] – наследует все поля и методы.  
- Конструктор с параметрами позволяет создавать объект сразу с заполненными данными.  
- Можно использовать для сериализации/десериализации JSON.
