  
## Purpose  
Хранит **данные транспорта**, получаемые из API.  
Расширяет [[TransportBasicDto]].  
  
---  
  
## Fields  
  
| Name | Type | Description |  
| --------------- | ------------------------ | ------------------------------- |  
| id | Long | Уникальный идентификатор транспорта |  
| transportModel | String | Модель транспорта |  
| transportNumber | String | Регистрационный номер/гос. номер |  
| typeOf | [[type_of_transports]] | Тип транспорта |  
  
---  
  
## Business Meaning  
  
- Используется для **получения информации о транспорте** в API.  
- Позволяет отслеживать и обрабатывать данные о транспорте, связанном с экипажем или сотрудником.
